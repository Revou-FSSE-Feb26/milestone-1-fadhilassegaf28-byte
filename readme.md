# Portfolio Website - Abdullah Fadhil

Website ini merupakan proyek portfolio pribadi yang saya buat menggunakan HTML dan CSS sebagai bagian dari proses belajar Web Development.

## Tentang Website

Website ini berisi informasi mengenai:
- Profil diri
- Kemampuan dan minat
- Proyek yang pernah saya kerjakan
- Form kontak

Tujuan dari website ini adalah untuk mendokumentasikan proses belajar saya dalam bidang teknologi dan pengembangan website.

## Fitur Website

- Halaman Home
- Halaman About Me
- Halaman Project
- Halaman Contact
- Desain sederhana menggunakan CSS

## Teknologi yang Digunakan

- HTML
- CSS

## Daftar Proyek

### 1. Analisis Strategi Digital Marketing
Proyek ini merupakan tugas mata kuliah Digital Marketing yang membahas analisis strategi pemasaran digital pada media sosial.

### 2. Analisis Sistem Informasi
Proyek ini berfokus pada analisis sistem informasi dalam membantu pengambilan keputusan dalam organisasi.

### 3. Website Portfolio HTML
Website portfolio sederhana yang dibuat untuk menampilkan informasi pribadi dan proyek yang telah dikerjakan.
